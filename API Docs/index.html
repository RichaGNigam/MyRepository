Pushfyi - V 1.4

Introduction

Welcome to PushFYI, your Real-time WebSocket gateway. Pushfyi allows you to rapidly build, test and deploy real-time features for your applications.

Please go through the Release notes and Changelog if you are upgrading from an older version.

v1.4Ubuntu 64 Bit

Installation Information

File location Description
/usr/local/bin/pushfyi PushFYI WebSocket Gateway
/usr/share/pushfyi/documentation Location of this documentation
/etc/pushfyi/pubnet.conf PushFYI configuration file.
/var/log/jarvis.log PushFYI log file
/etc/pushfyi/keys SSL Certificate and key file
